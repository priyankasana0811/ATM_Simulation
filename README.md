# ATM Simulation Project

## Overview

The ATM Simulation Project is a Python-based mini project that mimics the basic functionalities of an Automated Teller Machine (ATM). It provides a menu-driven interface where users can perform banking operations such as checking account balance, depositing money, and withdrawing money.

## Features

* Check account balance
* Deposit money into the account
* Withdraw money from the account
* Validate available balance before withdrawal
* User-friendly menu-driven interface
* Exit option to terminate the program

## Technologies Used

* Python

## Concepts Used

* Variables
* Input and Output
* Conditional Statements (if-elif-else)
* Loops (while)
* Arithmetic Operations

Through this project, I gained hands-on experience in Python programming, problem-solving, and implementing real-world scenarios using basic programming concepts.
